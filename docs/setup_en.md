# Daily Puzzles Setup Guide

## Required Software

- None

## Installation Procedures

No installation is required. Daily Puzzles is a Manual world, so setup is done through the Archipelago player settings page and the generated config file.

## Joining a MultiWorld Game

1. Open the player settings page and choose which puzzle sets you want enabled.
2. Set your preferred victory target with `Puzzles Required for Victory`.
3. Export the generated config file and join the multiworld using the Manual client.

## Multiplayer Manual

Different players can use different puzzle mixes. One player might run only LinkedIn puzzles, another only NYT puzzles.

## Game Troubleshooting

If a puzzle does not appear in your world, check that its category is enabled in the options and that your victory target does not exceed the number of active puzzle unlocks.
