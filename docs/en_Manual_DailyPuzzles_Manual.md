# Daily Puzzles

## What is Daily Puzzles?
Daily Puzzles is a Manual world built around everyday puzzle games from LinkedIn and the New York Times. Unlock items represent the individual puzzles you can complete, and locations track their clears.

## Where is the settings page?
The player settings page for this game is located <a href="../player-settings">here</a>. It contains all the options you need to configure and export a config file.

## What does randomization do to this game?
Archipelago randomizes the puzzle unlocks you place in your world and others. The exact set of checks depends on which puzzle categories you enable and how many puzzles you require for victory.

## What is the goal when randomized?
The default goal is to collect enough puzzle unlocks to satisfy your configured victory requirement.

## Which items can be in another player's world?
All of the puzzle unlocks you enable can be placed in another player's world.

## What if I want to disable some puzzles?
Turn off the corresponding puzzle category in the options. Disabled puzzles will not contribute unlocks or locations to the world.

## How do I install the mod for Daily Puzzles?
You don't. There is no mod. Progress is tracked manually while using the Manual client and its accompanying tracker.
